CSS BASIC KEY


What is css

CSS is the language we use to style a Web page.

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen or viewport.External stylesheets are stored in CSS files
  • External stylesheets are stored in CSS files
When a browser reads a style sheet, it will format the HTML document according to the information in the style sheet.
HOW TO ADD CSS IN HTML
Three Ways to Insert CSS
  1. Inline Css
  2. Internal Css
  3. External Css

Inline css

An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

We can add inline css according to below syntax

<p sytle="css property: property value;">

"In this above example we inserted a inline style attribute which gives background color as silver & text color as black "

Internal css

An internal style sheet may be used if one single HTML page has a unique style. The internal style is defined inside the <style> element, inside the head section.

We can add Internal Css according to below syntax

<head >
< style >
element OR selector(id,class & tag) {
css property: property value;
}
</ style >
< /head >

This paragraph is designed with the help of internal css

External css

With an external style sheet, you can change the look of an entire website by changing just one file! Each HTML page must include a reference to the external style sheet file inside the element, inside the head section.

An external style sheet can be written in any text editor, and must be saved with a .css extension. The external .css file should not contain any HTML tags.

We can add Eternal Css according to below syntax

<head >
< link="rel="stylesheet" href="mystyle.css"" >
< /head >

Here href is the refrence or we say the path if external css file.

This paragraph is designed with the External Css.
External css always have a extension with(".css").
External css file is very usefull for large document.
